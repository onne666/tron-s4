

# Tron Secure Engine™ — 钱包安全检测工具

## 概述
一个移动端优先的 Tron 区块链钱包安全检测 H5 应用，深色科技风格，Tron 红主色调，提供完整的连接→授权→检测→结果流程。

## 设计系统
- **主题**: 深色模式，深邃海军蓝背景 + Tron 红 (#FF0033) 强调色
- **字体**: Space Grotesk，科技感几何无衬线
- **材质**: 发光边框、内发光卡片、噪点纹理背景
- **动效**: 品牌曲线 cubic-bezier(0.25,1,0.5,1)，hover 上浮，active 压缩

## 页面与流程

### 1. 首页
- Logo + "钱包安全检测" 标题 + "实时链上风险分析" 副标题
- 核心大按钮「连接 Tron 钱包」，带发光效果
- 四个功能说明卡片（黑名单检测、风险地址识别、资金来源分析、地址行为分析）
- 底部信任背书文案 + "Tron Secure Engine™" 品牌

### 2. 钱包连接状态
- 未连接：显示连接按钮，支持 TronLink 等钱包
- 已连接：显示缩略地址 + TRX/USDT 余额
- 授权按钮「授权并开始检测」+ 用途说明
- 完整的 loading/成功/失败状态反馈

### 3. 检测过程页
- SVG 雷达扫描动画（红色扫描线360度旋转）
- 背景区块链节点网络图（低透明度）
- 动态文本轮播（正在分析交易记录→检测风险地址→分析资金来源→评估风险等级）
- 持续 3-6 秒

### 4. 安全结果页（无风险）
- 绿色盾牌 + 对勾图标
- 标题「该地址安全」+ 副标题「未检测到风险行为」
- 详细信息卡片（风险评分：低、资金来源：正常、地址行为：正常、关联地址：无风险）
- 按钮：重新检测 / 查看详细报告

### 5. 风险结果页（高风险）
- 红色警告图标 + 脉冲闪烁动效
- 标题「检测到高风险」
- 风险详情卡片列表（黑名单资金、高风险关联、行为异常、诱饵地址等）
- 红色风险评分进度条
- 安全建议 + 操作按钮

## 技术实现
- **Web3 集成**: TronWeb SDK 连接 TronLink 钱包（模拟检测逻辑，可对接真实 API）
- **TRC20 授权**: USDT 合约授权交互
- **动画**: Framer Motion 页面过渡 + CSS 动画（雷达扫描、脉冲发光）
- **状态管理**: React useState 管理连接/授权/检测流程状态
- **移动端优先**: 全程 max-w-md 居中，拇指友好的操作区域
- **地址校验**: Tron 地址格式验证
- **错误处理**: 连接失败、授权失败、检测超时等完整错误提示

## 组件结构
- `HomePage` — 首页与连接入口
- `WalletConnect` — 钱包连接与状态管理
- `AuthorizationView` — USDT 授权流程
- `ScanningView` — 检测过程动画
- `SafeResultView` — 安全结果展示
- `RiskResultView` — 风险结果展示
- `RadarAnimation` — 雷达扫描 SVG 动画
- `NetworkBackground` — 区块链节点背景
- `FeatureCard` — 功能说明卡片
